# üëê Request access to LLMs in Bedrock 

In this lab, we will use chat completion LLMs available on Amazon Bedrock. But first, you need to enable the models.

Once you have joined the workshop, navigate to the AWS console by clicking **Open AWS console** in the sidebar under **AWS account access**.

<Screenshot url="https://us-west-2.console.aws.amazon.com/event/dashboard" src="img/screenshots/30-aws/2-bedrock/1-nav-console.png" alt="Navigate to AWS Console" />

Once logged into the console AWS console, navigate to Amazon Bedrock.

<Screenshot url="https://us-west-2.console.aws.amazon.com/console/home" src="img/screenshots/30-aws/2-bedrock/2-nav-bedrock.png" alt="Navigate to Bedrock" />

In the Bedrock console, scroll down to **Bedrock configurations** in the sidebar and click **Model access**

<Screenshot url="https://us-west-2.console.aws.amazon.com/bedrock/home" src="img/screenshots/30-aws/2-bedrock/3-model-access.png" alt="Get model access" />

Under **Base models**, click on **Available to request** against the _Claude 3 Sonnet_ model. In the modal that appears, click **Request model access**.

<Screenshot url="https://us-west-2.console.aws.amazon.com/bedrock/home" src="img/screenshots/30-aws/2-bedrock/4-enable-model.png" alt="Enable model" />

Scroll to the bottom of the page and click **Next**. On the following page, click **Submit** to submit the model access request. 

It takes a few seconds to provision access. Once access has been granted you should see **Access granted** against the model name.

<Screenshot url="https://us-west-2.console.aws.amazon.com/bedrock/home" src="img/screenshots/30-aws/2-bedrock/5-access-granted.png" alt="Access granted" />

That's it! You are now ready to use the _Claude Sonnet 3_ model to build an AI agent.
